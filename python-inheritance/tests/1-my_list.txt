========================
MyList class tests
========================

Importing the MyList class
--------------------------

>>> MyList = __import__('1-my_list').MyList

Creating an empty list
----------------------

>>> my_list = MyList()
>>> print(my_list)
[]

Adding elements to the list
---------------------------

>>> my_list.append(1)
>>> my_list.append(4)
>>> my_list.append(2)
>>> my_list.append(3)
>>> my_list.append(5)
>>> print(my_list)
[1, 4, 2, 3, 5]

Printing the sorted list (without modifying the original)
---------------------------------------------------------

>>> my_list.print_sorted()
[1, 2, 3, 4, 5]

Verifying the original list is unchanged
----------------------------------------

>>> print(my_list)
[1, 4, 2, 3, 5]

Testing with another list
-------------------------

>>> another_list = MyList([10, -2, 7, 3])
>>> print(another_list)
[10, -2, 7, 3]
>>> another_list.print_sorted()
[-2, 3, 7, 10]
>>> print(another_list)
[10, -2, 7, 3]
